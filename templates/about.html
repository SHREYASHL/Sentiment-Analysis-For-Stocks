SENTIMENT-DRIVEN STOCK ANALYSIS & PORTFOLIO OPTIMIZATION

Technology Used

Flask

A lightweight WSGI web application framework used for building the backend API.

Tailwind CSS

A utility-first CSS framework for designing a responsive and modern user interface.

Sentiment Analysis API

An API service to analyze and interpret the sentiment of news and market data.

Moneycontrol News Web Scraping

A web scraping solution to gather the latest stock market news from Moneycontrol.

Deployed on Vercel

Hosting and deployment of the web application on Vercel for scalable and global access.
